## Installation
1. Download autoscript
```bash
wget https://raw.githubusercontent.com/skynetcenter/ubuntu-vpn/main/autoscript.sh
```

2. Execute and run autoscript
```bash
chmod +x autoscript.sh && autoscript.sh
```
